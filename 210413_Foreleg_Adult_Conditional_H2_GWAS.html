Adult Foreleg Length - Heritability estimate and GWAS Conditional Analysis

Caelinn James

02 April 2021

Written for R 4.0.2, last updated 31/07/21

This is a follow on from the adult foreleg length heritability estimate and GWAS script. The analysis identified peaks on chromosome 19 and 16. Here, we fit SNPs s74894.1 and s22142.1 to see how much of the variation they accounts for and to see if there are any close significant SNPs that are obscured by the original peaks.

#!/bin/sh
#$ -cwd
#$ -l h_rt=2:00:00
#$ -pe sharedmem 1

#$ -m baes
. /etc/profile.d/modules.sh

module load igmm/apps/dissect/1.15.2c
module load igmm/apps/plink/1.90b4

i=$_TASK_ID

plink --bfile 50K/Conditional_50K_Geno --sheep --not-chr ${i},X --keep AdultForelegPhenotypes.txt --make-bed --out 50K/NoChr${i}

mpirun -np 8 dissect.mpich --make-grm --bfile 50K/NoChr${i} --out 50K/NoChr${i}_GRM

echo "50K/Chr${i} 50K/NoChr${i}_GRM" > 50K/GWAS_GRM${i}.txt

plink --bfile 50K/Conditional_50K_Geno --sheep --chr ${i} --keep AdultForelegPhenotypes.txt --make-bed --out 50K/Chr${i}

if [[ -f 50K/NoChr${i}_GRM.badsnps ]]
then
plink --bfile 50K/NoChr${i} --sheep --exclude 50K/NoChr${i}_GRM.badsnps --make-bed --out 50K/NoChr${i}_2

mpirun -np 8 dissect.mpich --make-grm --bfile 50K/NoChr${i}_2 --out 50K/NoChr${i}_GRM

fi

mpirun -np 8 dissect.mpich --gwas --bfile-grm-list 50K/GWAS_GRM${i}.txt --pheno AdultForelegPhenotypes.txt --pheno-col 2 --covar Assoc.txt --qcovar QAssoc.txt --random-effects ForelegRandom.txt --random-effects-cols 1 2 3 --out 50K/AdultForeLeg_50K_GWAS_Conditional_Chr${i}

if [[ -f 50K/AdultForeLeg_50K_GWAS_Conditional_Chr${i}.badsnps ]]
then
plink --bfile 50K/Chr${i} --sheep --exclude 50K/AdultForeLeg_50K_GWAS_Conditional_Chr${i}.badsnps --make-bed --out 50K/Chr${i}_2

echo "50K/Chr${i}_2 50K/NoChr${i}_GRM" > 50K/GWAS_GRM${i}.txt

mpirun -np 8 dissect.mpich --gwas --bfile-grm-list 50K/GWAS_GRM${i}.txt --pheno AdultForelegPhenotypes.txt --pheno-col 2 --covar Assoc.txt --qcovar QAssoc.txt --random-effects ForelegRandom.txt --random-effects-cols 1 2 3 --out 50K/AdultForeLeg_50K_GWAS_Conditional_Chr${i}
fi

files = list.files(pattern="AdultForeLeg_50K_GWAS_Conditional_Chr",recursive=TRUE)

SNPresults = do.call(rbind, lapply(files, function(x) read.table(x,header=T)))
SNPresults$Effect <- SNPresults$STDEV*SNPresults$STDEV*SNPresults$BETA*SNPresults$BETA

bimFile <- read.table("C:/Users/s1944058/Documents/Sheep/50KSNP/Plates_1to87_QC3.bim",header=F)

GWAS50KResults <- merge(bimFile,SNPresults[,c("SNP","PV","Effect"),],by.x="V2",by.y="SNP",all.x=FALSE,all.y=TRUE)
colnames(GWAS50KResults) <- c("SNP","Chr","V3","Pos","Ref","Minor","PV","Effect")
GWAS50KResults$Chr <- as.numeric(GWAS50KResults$Chr)
par(mfrow=c(1,2))
manhattan(GWAS50KResults,chr="Chr",bp="Pos",p="PV",snp="SNP", main = "Adult foreleg length Manhattan plot (50K)", genomewideline = -log10(0.05/20082),suggestiveline = FALSE)
qq(GWAS50KResults$PV,main="Adult foreleg length QQ plot (50K)")

SNPs <- GWAS50KResults[which(GWAS50KResults$PV < (0.05/20082) ),c("Chr","Pos","SNP","PV","Effect")]
